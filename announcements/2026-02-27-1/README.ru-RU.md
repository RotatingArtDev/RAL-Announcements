# Небольшое обновление лаунчера 2.0.1 вышло!!

## Обзор обновления
- Версия 2.0.1 в основном сосредоточена на исправлениях совместимости, усилении диагностики и улучшении процессов сборки/релиза.
- Исправлены сбои инициализации CoreCLR на некоторых устройствах (особенно на части моделей Xiaomi), а также улучшен экспорт crash-логов.
- Обновлены графические зависимости (FNA3D и MobileGlues) и добавлены новые возможности наблюдения за производительностью.

## Исправления и совместимость
- Исправлены проблемы загрузки модов SMAPI в ряде сценариев.
- Добавлены и доработаны hooks, связанные с `pthread_condattr_setclock`, для повышения совместимости .NET runtime на отдельных устройствах.
- Добавлена обработка совместимости и логика повторной попытки при ошибках инициализации CoreCLR на некоторых устройствах Xiaomi.
- Улучшены взаимодействие со страницей сбоя и сбор диагностической информации для более полного расследования проблем.

## Производительность и диагностика
- Усилены инструменты тестирования/диагностики производительности GPU для более наглядного выявления узких мест рендеринга.
- Улучшен процесс экспорта логов для удобной отправки отчётов и анализа воспроизведения проблемы.

## Обновления зависимостей и runtime
- MobileGlues обновлён до версии 1.3.4.
- FNA3D обновлён и переключён на подходящую ветку, с дополнительной поддержкой для profiling-сценариев.

## Инженерия и релиз
- Добавлен workflow автоматической сборки APK в GitHub Actions для повышения эффективности и воспроизводимости релизов.
- Номер версии обновлён до 2.0.1.
